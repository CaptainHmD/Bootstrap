for installation{
    select your path then
    write in git terminal:    npm i bootstrap@5.2.0 
    then you have to link it

    on css you have to link this  file
    bootstrap.min.css
    and the path : /node_modules/bootstrap/dist/css/bootstrap.min.css


    on javascript you have to link this file
    bootstrap.bundle.min.js
    and the path : /node_modules/bootstrap/dist/js/bootstrap.bundle.min.js

    or use cdn
        link this in your html file 
         (this is for css)
        <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.2.0/dist/css/bootstrap.min.css" rel="stylesheet"
        integrity="sha384-gH2yIJqKdNHPEq0n4Mqa/HGKIhSkIHeL5AyhkYV8i59U5AR6csBvApHHNl/vI1Bx" crossorigin="anonymous">

        (this is for javascript)
        <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.2.0/dist/js/bootstrap.bundle.min.js"
        integrity="sha384-A3rJD856KowSb7dwlZdYEkO39Gagi7vIsF0jrRAoQmDKKtQBHUuLZ9AsSv4jD4Xa"
        crossorigin="anonymous"></script>
